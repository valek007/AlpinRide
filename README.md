# Alpine Ride 🏔️🚌

**Alpine Ride** is a front-end educational project developed as part of the **Wonders** module.  
The goal is to build a **responsive landing page** for a bus ticket service in the Alps, following a provided design and technical requirements.

---

## Project Phases

The project is structured in two main phases:

1. **Part 1 – HTML and CSS**
   - **Part 1.1:** Basic page structure, header, hero section, SCSS setup, desktop layout
   - **Part 1.2:** Additional static components (achievements, gallery, download section)
2. **Part 2 – JavaScript Functionality**

---

## 📌 Current Status – Part 1.2

The project currently focuses on **layout, structure, and additional static components**.  

### ✅ Completed (Part 1.1 & 1.2)

**Part 1.1 – Core Layout:**
- HTML page structure
- Header with logo and navigation menu
- Hero section:
  - Title and description
  - Trip type selection (Round Trip / One Way)
  - Passenger counter UI (static)
- SCSS architecture using component files
- Base styles, reset, header, and main section styling
- Desktop layout (1440px)
- Sass compilation setup via npm scripts

**Part 1.2 – New Static Components:**
- Achievements section
- Gallery section
- Download section with images

### Not Implemented Yet

- JavaScript functionality:
  - Dynamic passenger counter
  - Form validation
  - Autocomplete inputs
  - Date picker
- Responsive layouts for tablet and mobile
- Interactive elements and animations
